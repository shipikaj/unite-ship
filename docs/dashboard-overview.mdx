# Dashboard Overview

The Unite dashboard is the centralized workspace for managing data sources, destinations, pipelines, operational monitoring, users, billing, and account settings.

The platform is organized into dedicated operational sections that help teams manage the complete data workflow lifecycle from a single interface.

---

# Dashboard Structure

The Unite workspace is divided into the following primary areas:

- Home
- Data Sources
- Destinations
- Pipelines
- Alerts
- Account Settings
- Profile Menu

Each section supports a different stage of the data integration and monitoring workflow.

---

## Home

The **Home** page provides a high-level overview of workspace activity and operational health.

This section includes:
- Data sync overview
- Recent account activity
- Team visibility
- Recommended integrations
- Operational insights

The Home page acts as the primary monitoring layer for the workspace.

### Quick Actions

The top action bar provides quick access to commonly used workflows, including:
- Connect data source
- Connect destination
- Add new user

These actions help accelerate setup and operational tasks directly from the dashboard.

---

## Data Sources

The **Data Sources** section is used to manage platforms where data is extracted from.

Users can:
- Connect source accounts
- Monitor integration status
- Review connected accounts
- Reconnect expired integrations
- Remove unused sources

Each source includes operational details such as:
- Account information
- Source type
- Status
- Creation timestamp

---

## Destinations

The **Destinations** section manages systems where synchronized data is delivered.

This area allows users to:
- Configure destination systems
- Monitor destination health
- Manage warehouse integrations
- Review incomplete configurations

Supported destination types may include:
- BigQuery
- Amazon Redshift
- Data Studio

---

## Pipelines

The **Pipelines** section manages synchronization workflows between connected sources and destinations.

Users can:
- Create and manage pipelines
- Monitor sync activity
- Review pipeline history
- Identify failed or incomplete workflows
- Track synchronization status

Pipeline statuses may include:
- Active
- Paused
- Incomplete
- Error

---

## Alerts and Notifications

The **Alerts** section provides visibility into operational issues and workspace events.

This section may include:
- Pipeline alerts
- Integration failures
- System notifications
- Subscription and usage alerts

Users can also manage notification preferences and operational alert settings directly from this area.

The notification center in the top navigation bar provides quick access to:
- Recent activity
- Pipeline events
- Workspace updates
- Operational notifications

---

## Account Settings

The **Account Settings** section provides access to workspace-level configuration and administrative settings.

This section includes:
- User Management
- Billing
- Plans

These settings help administrators manage workspace access, subscriptions, and operational configuration.

---

### User Management

The **User Management** area allows administrators to:
- Invite users
- Assign workspace roles
- Remove users
- Manage permissions
- Control workspace collaboration

New users can be added directly through the side drawer invite flow.

---

### Billing

The **Billing** area provides visibility into:
- Workspace billing information
- Payment details
- Invoice history
- Subscription charges

Billing access may vary depending on workspace permissions.

---

### Plans

The **Plans** area allows users to:
- Review active subscriptions
- Compare available plans
- Monitor usage limits
- Upgrade workspace plans

Learn more:
- [Plans](../pricing-and-plans/plans)
- [Billing & Quotas](../pricing-and-plans/billing-and-quotas)

---

## Profile Menu

The profile menu is accessible from the top-right corner of the dashboard and provides quick access to user-level actions and support resources.

This section may include:
- User Profile
- Support Tickets
- Logout

Users can manage personal profile settings and access support resources directly from this menu.

---

## Search and Navigation

The global search bar allows users to quickly search across:
- Pipelines
- Sources
- Destinations
- Users

This helps improve navigation and operational visibility across larger workspaces.

---

## Book a Demo

The **Book a Demo** option allows users to schedule sessions with the Unite team for:
- Product walkthroughs
- Onboarding assistance
- Integration guidance
- Enterprise discussions
- Technical consultation

This option is accessible directly from the top navigation area.

---

## Support and Help

Support options are accessible from the dashboard and profile menus.

Available support options may include:
- Technical support
- Billing support
- Documentation access
- Feature requests
