# Prebuilt Reports

Unite provides a collection of predefined Google Analytics 4 reports for common analytics and reporting requirements. Each report includes a predefined set of dimensions and metrics, allowing you to create pipelines without manually configuring report fields.

Use a prebuilt report when the available field set meets your reporting requirements. Otherwise, create a custom report by selecting the required dimensions and metrics.
