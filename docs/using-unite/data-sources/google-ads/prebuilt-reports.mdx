# Prebuilt Reports

Unite includes a collection of predefined Google Ads reports for common advertising and campaign reporting requirements. Each report includes a predefined set of segments, metrics, and related fields, allowing you to start syncing data without manually configuring report fields.

Use a prebuilt report when the available field set meets your reporting requirements. Otherwise, create a custom report by selecting the required fields and metrics.
