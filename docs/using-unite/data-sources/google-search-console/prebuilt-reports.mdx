# Prebuilt Reports

Unite provides a collection of predefined Google Search Console reports for common SEO reporting requirements. Each report includes a predefined set of dimensions and metrics, allowing you to configure a pipeline without manually selecting report fields.

Use a prebuilt report when the available field set meets your reporting requirements. Otherwise, create a custom report by selecting the required dimensions and metrics.

---

## Available reports

| Report | Aggregation |
|---------|-------------|
| Keyword & Page Report | By page |
| Keyword & Site Report (By Page) | By page |
| Keyword & Site Report (By Site) | By property |
| Page Report | By page |
| Site Report (By Page) | By page |
| Site Report (By Site) | By property |

---

## Keyword & Page Report

**Report name:** `keyword_page_report`

Returns search performance metrics for each keyword and landing page combination. This is the most granular Search Console report available in Unite.

| Type | Field | Label |
|------|-------|-------|
| **Dimension** | `date` | Date |
| **Dimension** | `query` | Query |
| **Dimension** | `page` | Page |
| **Dimension** | `country` | Country |
| **Dimension** | `device` | Device |
| **Metric** | `clicks` | Clicks |
| **Metric** | `impressions` | Impressions |
| **Metric** | `ctr` | CTR |
| **Metric** | `position` | Average Position |

**Aggregation:** By page
