# Connect Data Studio

This guide explains how to use **Data Studio** as a destination in Unite.

Once your pipeline is configured, Unite automatically generates Data Studio reports that can be accessed directly from the pipeline.

---

## Before you begin

Ensure the following requirements are met:

- A pipeline has been created with **Data Studio** selected as the destination.
- You have access to a Google account with Data Studio permissions.

---

## Create a pipeline

Before opening Data Studio, you must create a pipeline in Unite.

1. Navigate to **Pipelines**.
2. Click **Create Pipeline**.
3. Select your preferred **Data Source**.
4. Choose **Data Studio** as the destination.
5. Complete the remaining pipeline configuration.
6. Save the pipeline.

Once the pipeline is successfully created, you can access the generated Data Studio report.

---

## Open Data Studio

You can open your Data Studio report directly from Unite in two ways.

### Option 1 - From the Status tab

1. Open the required pipeline.
2. Navigate to the **Status** tab.
3. Click **Open Data Studio**.

The associated Data Studio report opens in a new browser tab.

---

### Option 2 - From the Schema tab

1. Open the required pipeline.
2. Navigate to the **Schema** tab.
3. Locate the table or report you want to view.
4. Click **Open Report**.

The corresponding Data Studio report opens for the selected dataset.

---

## Verify the connection

After opening the report, verify that:

- The report loads successfully.
- The connected datasets are available.
- Tables and fields are displayed correctly.
- Visualizations reflect the latest synchronized data.

If data is not available, ensure the pipeline has completed a successful synchronization.

---

## Next step

Once your report is available in Data Studio, you can customize it by adding charts, filters, scorecards, and other visualizations to create dashboards tailored to your reporting needs.
