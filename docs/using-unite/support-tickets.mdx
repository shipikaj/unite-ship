# Support & Tickets

Unite provides multiple ways to get help depending on the type of assistance required.

Use the Help menu to contact the appropriate team, then track your requests through the Support Tickets section.

---

# Getting Help

Select the **Help** icon in the top navigation to access support resources.

Available options include:

### Contact Billing Support

Use billing support for:

- Subscription questions
- Invoice requests
- Payment issues
- Plan upgrades
- Plan downgrades
- Renewal inquiries

---

### Contact Technical Support

Use technical support for:

- Pipeline failures
- Authentication issues
- Source connection problems
- Destination connection issues
- Data synchronization concerns
- Platform errors

---

### Documentation

Browse Unite documentation for:

- Setup guides
- Connector documentation
- Pipeline configuration
- Billing information
- Troubleshooting articles

Many common questions can be resolved directly through documentation.

---

### Request a Feature

Submit enhancement requests for:

- New connectors
- Platform improvements
- Workflow enhancements
- Product suggestions

Feature requests are reviewed by the Unite product team.

---

# Tracking Support Requests

All submitted support requests are available in the **Support Tickets** section.

Support Tickets allows you to:

- Review active requests
- Monitor ticket progress
- View previous conversations
- Check resolution status

---

# Ticket Status

### Open

Your request has been received and is awaiting review.

### Pending

The support team is actively working on the request. Additional information may be requested during this stage.

### Closed

The issue has been resolved and the request has been completed.

---

# Best Practices

To help us resolve requests quickly:

- Include relevant error messages
- Specify affected pipelines or connectors
- Provide screenshots when available
- Describe recent configuration changes

Providing complete information helps reduce investigation time and speeds up resolution.
