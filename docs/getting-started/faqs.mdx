# FAQs

Find answers to common questions about using Unite.

---

<details>
  <summary>Do I need coding skills to use Unite?</summary>

  No. Unite provides a no-code pipeline builder.
</details>

<details>
  <summary>Can I connect multiple data sources?</summary>

  Yes. You can connect multiple sources and send data to one or more destinations.
</details>

<details>
  <summary>What if I can’t find a connector I need?</summary>

  We regularly add new connectors. If something is missing, you can request it and we will prioritise it.
</details>

<details>
  <summary>What happens when my free trial ends?</summary>

  If no payment method is added, your pipelines will pause. You can resume anytime by upgrading.
</details>

<details>
  <summary>Is my data secure?</summary>

  Yes. Unite uses industry-standard security practices, including encryption and access controls.
</details>

<details>
  <summary>Can I stop pipelines anytime?</summary>

  Yes. You can pause or delete pipelines at any time.
</details>

<details>
  <summary>What makes Unite different from other ETL tools?</summary>

  Unite provides a no-code interface, making it suitable for both non-technical users or engineers.
</details>
