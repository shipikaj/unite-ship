# Monitor Usage

Track your usage to avoid hitting limits.

Learn why deleted pipelines still count: [Billing & Quotas](./billing-and-quotas)

---

## What You Can See

- Pipelines created  
- Active pipelines  
- Deleted pipelines  
- Data sources  
- Accounts  

---

## Usage Behavior

### Deleted Pipelines

- Still visible in usage logs  
- Include timestamps (created, modified, deleted)  
- Count toward usage in the billing cycle  

---

## Alerts

- Warning when approaching limits  
- Notifications when limits are reached  

---

## Best Practices

- Avoid unnecessary pipeline creation  
- Reuse pipelines where possible  
- Monitor usage regularly
